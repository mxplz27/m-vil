## 4. Descripción del proyecto
Este proyecto fue desarrollado en el lenguaje de programación Dart. El programa permite registrar la información básica de un aprendiz, ingresar tres notas, calcular el promedio y realizar diferentes validaciones, como verificar si aprobó, si es mayor de edad, si está activo, si puede continuar con su proceso de formación y si requiere apoyo.

## 5. Objetivo de la actividad
Aplicar los conceptos básicos del lenguaje Dart mediante el desarrollo de un programa de consola que utilice variables, tipos de datos, operadores, estructuras de decisión y comentarios para resolver un problema sencillo.

## 6. Temas trabajados
- Variables (`var`, `final` y `const`).
- Tipos de datos (`String`, `int`, `double` y `bool`).
- Entrada y salida de datos.
- Operadores aritméticos.
- Operadores relacionales.
- Operadores lógicos.
- Estructuras condicionales (`if` y `else`).
- Comentarios en Dart.
- Cálculo del promedio.
- Validación de información.

## 7. Instrucciones para ejecutar el programa

1. Abrir el proyecto en Visual Studio Code.
2. Abrir la terminal.
3. Ejecutar el siguiente comando:

```bash
dart run
```

4. Ingresar los datos solicitados por el programa.
5. Observar los resultados mostrados en la consola.
